# Проект React - Каталог рецептов  
**React-приложение для просмотра и поиска блюд на основе локальных данных**

Учебный проект по курсу веб-разработки. Реализован простой, но полноценный SPA-каталог рецептов с использованием React.

---

## Описание  
Приложение отображает список блюд со следующей информацией:
- Название
- Краткое описание
- Кухня (страна/традиция)
- Примерное время приготовления

Все данные хранятся локально в JavaScript-массиве — **бэкенд и интернет не требуются**.

---

## Основные возможности
- Просмотр списка рецептов
- Поиск по названию и описанию в реальном времени
- Адаптивный интерфейс (работает на ПК и мобильных устройствах)
- Минималистичный и читаемый дизайн

---

## Технологии
- **React** (функциональные компоненты, хуки: `useState`)
- **Чистый CSS** (без сторонних библиотек)
- **Локальные данные** (`src/data/recipes.js`)
- Совместимость: современные браузеры (Chrome, Firefox, Edge)
